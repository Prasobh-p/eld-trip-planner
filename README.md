# ELD Trip Planner

## Overview
Full-stack app that calculates trip routes and generates ELD-style daily logs.

## Tech Stack
- Django REST Framework
- React (Vite)
- OpenRouteService API
- Leaflet + OpenStreetMap

## Assumptions
- Property-carrying driver
- 70 hrs / 8 days
- Max 11 hrs driving per day
- 1 hr pickup and dropoff
- Fuel every ~1000 miles
- No adverse driving conditions

## Setup
- Backend: add ORS_API_KEY in .env
- Frontend: npm install && npm run dev
